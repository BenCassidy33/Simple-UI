Quick Start
Introduction
Simple UI is a clean, lightweight way to build elegant user interfaces without installing large libraries or hundreds of packages. It provides custom HTML elements that work like standard tags but include additional built-in functionality, making UI development simpler and more expressive.
Motivation
Many component libraries rely on frameworks like React or require additional tooling to function. Simple UI aims to offer similar flexibility using only native browser features. All components are delivered as custom HTML tags with optional attributes, made available through a CDN. See Quick Start for an example. Quick Start for an example.
